# Industrial-Equipment-Downtime-Analysis
Interactive Tableau Dashboard for analyzing industrial equipment downtime across multiple factories.
Industrial Equipment Downtime Analysis Dashboard
Project Overview

This Tableau dashboard analyzes equipment downtime across multiple manufacturing factories. The dashboard helps identify which device types contribute the most downtime and compares downtime across factory locations.

Objective
Monitor equipment downtime
Identify high-maintenance devices
Compare factory performance
Support maintenance decision-making
